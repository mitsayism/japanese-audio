# japanese-audio
# Japanese Audio Storage 🎧

Kho chứa file âm thanh (mp3) chuyên dụng cho ứng dụng học tiếng Nhật. 
Kho này được sử dụng làm CDN tĩnh thông qua jsDelivr để tối ưu tốc độ tải web.

## Cấu trúc đường dẫn CDN:
`https://cdn.jsdelivr.net/gh/<tên_tài_khoản_github_của_bạn>/<tên_repo>@main/<tên_file>.mp3`
